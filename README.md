v0.14alpha -- 29 Feb 2016

- Now in git.
- hipe_loc data type introduced for more intuitive use of locations.
- Provided ability to load background images for extant elements.
- hiped supports command line parameter: hiped --sockfile for creating a
  socket file with a non-default name.
- hiped now also supports command line parameter --keyfile for placing
  the top-level keyfile in a custom location.
