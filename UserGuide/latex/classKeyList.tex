\doxysubsection{Detailed Description}
Each client, and the global environment, holds a \mbox{\hyperlink{classKeyList}{Key\+List}} object. It indexes a list of randomly generated keys which a client needs in order to become parented by a particular host. A valid key can only be claimed once, afterwhich it\textquotesingle{}s removed from the pool.
